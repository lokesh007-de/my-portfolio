## ✨ Features

- 🎬 **Dark Cinematic Design** — Inspired by professional creative portfolios
- ⚡ **Framer Motion Animations** — Smooth scroll-triggered & page load animations
- 🖱️ **Custom Cyan Cursor** — Interactive cursor with lag effect
- 📱 **Fully Responsive** — Mobile, tablet & desktop optimized
- 🎨 **Glassmorphism Navbar** — Blur backdrop on scroll
- 🔤 **Letter-by-Letter Animation** — Staggered name reveal on load
- 🌟 **Skill Bars Animation** — Animated on scroll into view
- 🗂️ **Project Timeline** — All GitHub projects with links
- 🃏 **Portfolio Cards** — Hover glow effect with tech stack tags

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/lokesh007-de/my-portfolio.git

# Navigate to project directory
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
